# Nebula Dashboard

Nebula is a next-generation analytics platform designed to make complex data intuition easy. By leveraging LLMs, it can interpret raw data and generate natural language insights alongside interactive visualizations.

## Key Features

- **Natural Language Querying**: Ask questions about your data in plain English.
- **Real-time Visualization**: Dynamic charts that update as data flows in.
- **Predictive Analytics**: Forecasting trends based on historical data.

## Technical Architecture

Built with Next.js for the frontend and a Python FastAPI backend for the AI processing.
