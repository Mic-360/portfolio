# Optimizing React Performance for 2024

Performance is user experience. In 2024, the React ecosystem offers powerful tools to ensure your applications remain snappy even as they grow in complexity.

## The Power of `useMemo` and `useCallback`

While not a silver bullet, proper memoization prevents unnecessary re-renders.

```tsx
import { useMemo } from 'react'

function ExpensiveComponent({ data }) {
  const processedData = useMemo(() => {
    return data.map((item) => item * 2).filter((item) => item > 10)
  }, [data])

  return <div>{processedData.join(', ')}</div>
}
```

## React Server Components (RSC)

RSCs allow you to run heavy logic on the server and send only the necessary HTML to the client, significantly reducing bundle size.

### Benefits of RSC

1.  **Zero Bundle Size**: Dependencies used on the server don't impact the client.
2.  **Direct Database Access**: Query your DB directly from your component.
3.  **Automatic Code Splitting**: Client components are automatically split.
