## The Stack (Because Minimalism is for Quitters)

The Holy Quadrinity of mobile dev enlightenment:

1. **Termux**: Your new home. It's a terminal emulator that makes your phone feel like the power tool it actually is (instead of just a camera for your brunch).
2. **Node.js/ Bun**: Because of course we're using JavaScript environments.
3. **Claude Code**: The paid AI god for people with salaries.
4. **OpenCode + Antigravity Auth**: The "freebie" AI god for the rest of us.

## Step 1: The Sacrifice (Termux Setup)

First, download Termux. Don't use the Play Store version; it's practically ancient history. Get it from F-Droid or GitHub.

Open it up and run the ritual:

```bash
pkg update && pkg upgrade
```

Install Ubuntu(coz i like it don't tell me some weird distro name):

```bash
pkg install proot-distro
proot-distro install ubuntu
proot-distro login ubuntu
```

Update your system:

```bash
apt update && apt upgrade -y
apt install curl git nodejs npm -y
```

Now your phone is officially a development machine. You can finally tell people you "work in terminal" when you're actually at Starbucks.

## Step 2: Choosing Your AI Deity

Two paths. One has a monthly billing cycle, the other has character.

### Path A: The "I Have a Budget Line for This" Option (Claude Code)

If you're okay paying Anthropic to technically do your job while you sip coffee:

```bash
curl -fsSL https://claude.ai/install.sh | bash
claude --remote
```

Done. Go back to your $15 avocado toast.

### Path B: The "I Have Principles" Option (OpenCode + Antigravity Auth)

For those of us who prefer to keep money for important things (like RAM or high-quality coffee), we use **OpenCode**.

I found the antigravity plugin while responsibly not doom-scrolling and it genuinely simplified my life: no monthly AI subscription just to get an LLM to tell me why my regex is wrong.

#### Setting up OpenCode + Antigravity

First, install OpenCode:

Easiest way:

```bash
curl -fsSL https://opencode.ai/install | bash
```

Medium way:

```bash
npm install -g opencode-ai
```

Using Homebrew on macOS and Linux:

```bash
brew install anomalyco/tap/opencode
```

Now, login:

```bash
opencode auth login
```

**CRITICAL NOTE**: Android Termux doesn't support the fancy OpenCode UI. You're a terminal purist now. Use the `--web` flag or it'll just look like your screen is having a stroke.

```bash
opencode web --hostname [YOUR_TAILSCALE_IP] --port 4096
```

> **Note:** If things go wrong and you see an error after choosing google > antigravity, then you need to setup some env-vars.

```bash
export XDG_CACHE_HOME=\$HOME/.cache
export OPENCODE_SERVER_USERNAME="your_username"
export OPENCODE_SERVER_PASSWORD="your_password"
```

Still not working? Try this: `"plugin": ["opencode-antigravity-auth@beta"],`

## Step 3: Tailscale (The Packet Wormhole)

If you're running `opencode web`, you'll need a way for your browser to talk to that proot-distro Ubuntu instance without Android's sandbox throwing a tantrum. Enter **Tailscale**.

1. **Install the App**: Download Tailscale from the Play Store or F-Droid.
2. **Login**: Use the same account you use on your other machines.
3. **Turn it ON**: Flip that switch. You're now part of the mesh.

Find your phone's Tailscale IP (the one that looks like `100.x.y.z`). That is your new identity. Use it in the `opencode web` command so you can access your dev environment from your mobile Chrome, or even from your laptop if you're feeling extra lazy.

## Troubleshooting (Skill Issues)

If it doesn't work:

1. **Termux Sleep**: Android loves to kill background processes. Make sure you enable "Keep CPU awake" in the Termux notification or your session will die faster than a startup with no VC funding.
2. **Node Versions**: If `npm` acts up, `pkg reinstall nodejs-lts`.
3. **OpenCode UI**: I told you already, use `--web`. If you forgot, that's a classic skill issue.
